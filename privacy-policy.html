1. Introduction

Garment Me ("the App") is an AI-powered virtual try-on application for Android devices. We are committed to protecting your privacy and being transparent about how we handle your personal information. This policy explains our practices regarding data collection, use, and protection.

2. App Permissions

Required Permissions

  • Camera Access: To enable real-time virtual try-on experiences
  • Photo Library Access: To use existing images for virtual try-ons
  • Storage Access: To save customized try-on results (optional)

All permissions are used exclusively for app functionality. You can revoke permissions anytime through device settings.

3. Data Collection and Usage

  • Image Processing: We temporarily process body measurements and facial features to enable virtual try-ons
  • No Biometric Storage: All biometric data is processed in real-time and never stored
  • Usage Analytics: Collect anonymous usage data to improve app performance (device type, session length)
  • User Preferences: Store clothing preferences and size selections locally on your device

4. AI Processing

Our AI system processes images through these steps:

  1. Real-time body mapping using device camera input
  2. Virtual garment rendering using on-device machine learning
  3. Temporary caching of processed images (deleted after session)

No personally identifiable information is used to train our AI models.

5. Third-Party Services

  • Cloud Processing: Some complex garment simulations may use secure cloud servers (images are encrypted in transit)
  • Analytics Tools: We use Firebase Analytics with anonymized data collection
  • No Data Sharing: We do not sell or share personal data with advertisers or third parties

6. Security

  • Data Encryption: All network transmissions use TLS 1.2+ encryption
  • Local Storage: User preferences stored using Android Keystore protection
  • Automatic Deletion: All processed images are deleted within 24 hours

7. Children's Privacy

Our app complies with COPPA regulations:

  • Not directed at children under 13
  • Does not knowingly collect personal information from minors
  • Parents/guardians can request data deletion via our contact email

8. Changes to Policy

We may update this policy to reflect new features or regulations. Significant changes will be notified through in-app alerts. Continued use after updates constitutes acceptance of revised terms.
